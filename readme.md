## About The Project

<!-- [![Product Name Screen Shot][product-screenshot]](https://example.com) -->


Bone is an easy to use ShareX server that supports the popular application ShareX. It delivers an easy and simple way to upload images, gifs, videos and more to your own server. It also supports user management, so you can give your friends access to your server and upload from  domain.

Bone is heavily inspired by open source project [ShareX-Server](https://xbackbone.app/).
